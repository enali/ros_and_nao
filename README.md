# slides template

一个简单的制作幻灯片的项目模板

## 构建

```sh
$ git clone https://github.com/enali/slides_template project_name
$ cd project_name
$ git submodule update --init
$ bundle
$ write your slides content with editor
$ rake
$ open build/index.html # or open with your browser
```

## 约定

* 把图片放到`images`文件夹
* 把章节文档放到`sections`文件夹
